# MLB 2024 Season Batting Analysis

This project is using linear regression to try and understand which variables are important in predicting batting average and homeruns. 

## Usage
1. The "Reports" file have documents with a initial plan and final report.
2. The "Code" file has a quarto document with the corresponding CSV to run analysis.
3. When using the quarto document, you will have to download certain libraries to run the code.
